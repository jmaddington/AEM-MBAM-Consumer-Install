# Overview #
AEM component to *consumer* MalwareBytes.

# Usage #
Download and install this component in your AEM instance. Download the latest version of MBAM from https://www.malwarebytes.com/ and upload as mbam-setup.exe to the component.

The *actual installer* is NOT included with this component for licensing reasons.

This script will not restart the machine, the machine will need to be rebooted to complete the install.

# Building #
Download or fork, run repackage.bat and upload aem-component.cpt to AEM. You can also download the aem-component.cpt straight from this repository and install in your AEM instance.
